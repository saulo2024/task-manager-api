🚀 Task Manager API
API robusta para gerenciamento de tarefas, desenvolvida com foco em segurança e escalabilidade.

🛠️ Tecnologias (Technologies):

Node.js & Express: Core da aplicação.

MongoDB & Mongoose: Banco de dados NoSQL e modelagem de dados.

JWT (JSON Web Token): Autenticação segura.

Arcjet: Proteção contra robôs e controle de taxa (Rate Limiting).

🛡️ Segurança (Security):

Auth Middleware: Proteção de rotas privadas.

Input Validation: Validação rigorosa de IDs e formatos de dados.
